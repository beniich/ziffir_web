# Checklist stand salon

## Matériel
- [ ] Comptoir d'accueil avec kakémono roll-up (3m × 2m)
- [ ] Écran TV 65" 4K + support
- [ ] MacBook Pro M3 de démo (backup : second Mac)
- [ ] iPad Pro (démo mobile preview)
- [ ] Routeur 4G de secours (backup si WiFi salon saturé)
- [ ] Multiprises × 4, rallonges × 3
- [ ] Sac de câbles (HDMI, USB-C, Ethernet)
- [ ] Ruban adhésif, attaches câbles, scotch double-face
- [ ] Aspirateur à poussière (les stands sont sales)
- [ ] Essuies-tout, spray nettoyant écran

## Marketing
- [ ] 500 cartes de visite (3 personnes × ~150)
- [ ] 200 brochures A4 papier couché 300g (coût : 200€)
- [ ] 100 stylos gravés
- [ ] 50 câbles USB-C
- [ ] 1 rouleau de tickets de tombola
- [ ] 1 iPad pour scanner les badges visiteurs (app Badge Scanner)
- [ ] Oriflamme X (ex-Twitter) avec QR code vers /trial

## Personnel
- [ ] 3 polos brodés au logo Ziffir
- [ ] Badges nominatifs
- [ ] Eau + snacks pour l'équipe
- [ ] Cash pour les taxis (200€)
- [ ] Liste des RDV pré-programmés (imprimée)

## Démo
- [ ] Comptes de démo pré-configurés (3 hôtels fictifs)
- [ ] Vidéo de démo 60s en boucle sur l'écran
- [ ] Scénario démo rodé (10 min max)
- [ ] Backup : version offline de la démo (vidéos)

## Logistique
- [ ] Arrivée J-1 pour montage
- [ ] Briefing équipe J-1 : pitch, objections, objectifs
- [ ] Planning présence stand (rotations toutes les 2h)
- [ ] Objectifs chiffrés : 100 contacts, 20 démos, 5 trials

## Post-salon
- [ ] Débriefing J+1 (équipe)
- [ ] Email de remerciement J+2
- [ ] Suivi commercial J+3, J+7, J+14
- [ ] Article blog J+10
